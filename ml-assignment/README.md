# Trigram Language Model

This directory contains the core assignment files for the Trigram Language Model.

## How to Run

* **Clone the repository**
  ```git clone <repo-url>```
  ```cd ml-assignment```
* **Install Dependencies**
  ```pip install -r requirements.txt```
* **Train and generate the text manually to see the sample output**
  ```python src/ngram\_model.py```
* **Run unit tests**
  ```python -m pytest tests/test\_ngram.py```
